# Sample Cocktails App

Experiment app to work with RxSwift and MVVM. The app displays different cocktails and a list of their ingredients.

## Requirements

- Xcode 11.4
- iOS 13.2
- Swift 5

## Technologies

- Swift
- RxSwift
- RxCocoa
